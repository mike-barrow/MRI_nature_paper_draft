\section*{Code availability}\label{sec:code}

The released imaging and segmentation labels can be read and processed with the standard, openly available tools listed in the Usage Notes.
Manual segmentation was performed in 3D Slicer [TBD: version].
The DICOM headers were de-identified with the Clinical Trial Processor and the de-identification script supplied by the host repository, so that tooling is not released here.
The defacing configuration for the facial and pelvic regions, and the dcmqi-based conversion~\cite{Herz_2017} [TBD: version] of the working segmentation masks to the released DICOM-SEG objects, are candidates for public release [TBD: Felix/Michael at submission].
The training and evaluation code for the baseline model is [TBD: Michael at submission].
